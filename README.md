# Halloween Horror Nights Information

## Project Description
I created a web page that provides various information for Universal Orlando's Halloween Horror Nights

## Intended User or Player
This page is intended for guests of Halloween Horror Nights.

## User / Player Goal
After viewing my page, the user should have knowledge of the various offerings at Halloween Horror Nights, as well as dates and times and ticket information.

## Inspiration Interfaces
I was inspired by various other Halloween Event pages such as sea world's Howl O' Scream and Disney's Not so Scary Event. From these pages I was inspired to add images and calendars to my page.

## Design Choices
- Color: Black, white and red.
- Typography: Creepster.
- Visual hierarchy: the headings are much bigger than the paragraph text to allow users to differentiate between sections.
- Feedback states: my links change color depending on whether you are hovering over them.
- Responsive design: my text is able to wrap differently depending on the size of the screen.
